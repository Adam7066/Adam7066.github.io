Python-01：爬蟲-圖片下載

Aug 03, 2020

簡介

  • 這一篇將帶領大家透過Python的爬蟲自動化的下載圖片

安裝

  • 打開終端機，安裝下列套件
    • pip install requests
    • pip install BeautifulSoup4
    • pip install lxml
    • pip install selenium
  • 至 Chrome Driver 下載當前電腦中Chrome所對應版本的Driver

教學開始

  • 首先因為這次的主題為圖片下載，因此我們找了一個擁有大量圖片的漫畫網頁來進行教學。
  • 由於這個網頁的圖片是由JavaScript加載的，因此一開始我們使用Selenium來開啟網頁
    1from selenium import webdriver
2
3url = 'https://www.ohmanhua.com/13621/1/1.html'
4#使用crx插件
5chop = webdriver.ChromeOptions()
6chop.add_extension('Adblock-Plus_v3.8.4.crx')
7browser = webdriver.Chrome(options = chop)
8browser.implicitly_wait(10)
9browser.get(url)
    • 當然若不想顯示瀏覽器的視窗可以使用headless模式
    1chop.add_argument('--headless')  #規避google bug
2chop.add_argument('--disable-gpu')
  • 接下來使用BeautifulSoup4來分析頁面，並取得圖片網址，和共幾張圖片
1img_count = int(soup.find('select', {'class': 'mh_select'}).find_all('option')[-1].get('value'))

  • 最後便是下載圖片和關閉瀏覽器
1def download_img(img, num):
2    r = requests.get(img)
3    with open(save_url + str(num+1) + '.jpg', 'wb') as f:
4        f.write(r.content)

7    img = img_url + str(i + 1).zfill(m) + '.jpg'

9browser.quit()

完整程式碼

 1# -*- coding: UTF-8 -*-
 2from selenium import webdriver
 3from bs4 import BeautifulSoup
 4import requests
 5
 6def download_img(img, num):
 7    r = requests.get(img)
 8    with open(save_url + str(num+1) + '.jpg', 'wb') as f:
 9        f.write(r.content)
10    
11if __name__ == "__main__":  
12    save_url = './download/'
13    url = ''
14    chop = webdriver.ChromeOptions()
15    chop.add_extension('Adblock-Plus_v3.8.4.crx')
16    browser = webdriver.Chrome(options = chop)
17    browser.implicitly_wait(10)
18    browser.get(url)
19    soup = BeautifulSoup(browser.page_source, 'lxml')
20    img_count = int(soup.find('select', {'class': 'mh_select'}).find_all('option')[-1].get('value'))
21    img_url = soup.find_all('div', {'class': 'mh_comicpic'})[0].find('img').get('src')
22    if img_url[0] == '/':
23        img_url = 'https:' + img_url
24    m = len(img_url.rsplit('/', 1)[1].split('.')[0])
25    img_url = img_url.rsplit('/', 1)[0] + '/'
26    for i in range(img_count):
27        img = img_url + str(i + 1).zfill(m) + '.jpg'
28        download_img(img, i)
29    browser.quit()
