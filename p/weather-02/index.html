簡易氣象站-02

Jul 12, 2020

簡介

  • 這系列主題將帶領大家建立一個簡易的氣象觀測站，而這一小節將製作在Raspberry pi用Python讀取Arduino Serial的數值，並且製作一個簡易的UI實時顯示出數值。

硬體

  • 上一小節的成品
  • Raspberry pi * 1

教學開始

  1. 打開Raspberry pi的Terminal，並輸入 ls /dev/tty*
  2. 將Arduino的Usb接上Raspberry pi
  3. 再次於Terminal中輸入 ls /dev/tty*，比較兩次的差異，即可知Arduino使用的序列埠為何(例如此時為/ dev/ttyACM0，請依照情況自行修改)
  4. 於Terminal中輸入 python3 -m pip install pyserial
  5. 將以下程式碼建立成一個python檔，再以python3執行即可
 1# -*- coding: UTF-8 -*-
 2import serial
 3import tkinter as tk
 4
 5def getData():
 6    ser = serial.Serial('/dev/ttyACM0', 9600, timeout=1)
 7    ser.flush()
 8    while True:
 9        flag = False
10        if ser.in_waiting>0:
11            arduinoData = ser.readline().decode(encoding='utf-8', errors='ignore').rstrip().split(';')
12            if len(arduinoData)!=4 or arduinoData=='':
13                continue
14            for i in range(0,3):
15                if arduinoData[i]=='0':
16                    flag = True
17                    break
18            if flag:
19                continue
20            else:
21                temperature_value['text'] = arduinoData[0] + ' 度C'
22                humidity_value['text'] = arduinoData[1] + ' %'
23                pmat25_value['text'] = arduinoData[2] + ' ug/m^3'
24                wind_value['text'] = arduinoData[3] + ' 級'
25                break
26    window.after(1000, getData)
27
28if __name__=='__main__':
29    window = tk.Tk()
30    window.title('Weather')
31    window.geometry('500x250')
32    window.resizable(False, False)
33
34    header_label = tk.Label(window, text='環境監測', font=('Arial', 20), width=30, height=2, borderwidth=2, relief='solid')
35    header_label.pack(side=tk.TOP)
36
37    temperature_frame = tk.Frame(window)
38    temperature_frame.pack(side=tk.TOP)
39    temperature_label = tk.Label(temperature_frame, text='溫度: ', font=('Arial', 16))
40    temperature_label.pack(side=tk.LEFT)
41    temperature_value = tk.Label(temperature_frame, font=('Arial', 16))
42    temperature_value.pack(side=tk.LEFT)
43
44    humidity_frame = tk.Frame(window)
45    humidity_frame.pack(side=tk.TOP)
46    humidity_label = tk.Label(humidity_frame, text='濕度: ', font=('Arial', 16))
47    humidity_label.pack(side=tk.LEFT)
48    humidity_value = tk.Label(humidity_frame, font=('Arial', 16))
49    humidity_value.pack(side=tk.LEFT)
50
51    pmat25_frame = tk.Frame(window)
52    pmat25_frame.pack(side=tk.TOP)
53    pmat25_label = tk.Label(pmat25_frame, text='PM2.5: ', font=('Arial', 16))
54    pmat25_label.pack(side=tk.LEFT)
55    pmat25_value = tk.Label(pmat25_frame, font=('Arial', 16))
56    pmat25_value.pack(side=tk.LEFT)
57
58    wind_frame = tk.Frame(window)
59    wind_frame.pack(side=tk.TOP)
60    wind_label = tk.Label(wind_frame, text='風速: ', font=('Arial', 16))
61    wind_label.pack(side=tk.LEFT)
62    wind_value = tk.Label(wind_frame, font=('Arial', 16))
63    wind_value.pack(side=tk.LEFT)
64
65    getData()
66    window.mainloop()
