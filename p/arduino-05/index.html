#include <SoftwareSerial.h>

SoftwareSerial BT(8, 9);
char val;

void setup() {
    Serial.begin(9600);
    BT.begin(38400);
}

void loop() {
    if(Serial.available()){
        val = Serial.read();
        BT.print(val);
    }
    if(BT.available()){
        val = BT.read();
        Serial.print(val);
    }
}
  1. 連接腳位，並上傳程式碼，最後給HC-05供電前，先按住上面的按鈕，再提供電源，燈號將變成約兩秒一閃，及表示進入了AT命令模式
  2. 接下來打開序列埠監控視窗，將設定調成"9600 baud"和"NL與CR"，最後依需求輸入以下命令即可
  • AT -> 顯示OK表示連接成功
  • 查看韌體版本 -> AT+VERSION
  • 查看名稱 -> AT+NAME?
  • 修改名稱 -> AT+NAME=你要的名字
  • 查看密碼 -> AT+PSWD?
  • 修改密碼 -> AT+PSWD=你要的密碼
